OPI UITK Boot

This starter kit bootstraps the development of Angular 6+ applications leveraging the latest version of the UITK

UITK Boot offers the following key advantages
  • Built using angular cli for scaffolding [more information here]
  • Wrapped with Bootstrap 4 for easy grid layout [more information here]
  • Open source project, allowing for collaborative effort for improvements
  • Leveraging latest version of Optum UITK for reusable components
